# N Queens Problem Solver

This is a Python script to solve the N queens problem. The N queens puzzle is the challenge of placing N non-attacking queens on an NÃ—N chessboard.

## Requirements

- Allowed editors: vi, vim, emacs
- All files will be interpreted/compiled on Ubuntu 20.04 LTS using Python 3 (version 3.4.3)
- All files should end with a new line
- The first line of all your files should be exactly `#!/usr/bin/python3`
- Your code should use the PEP 8 style (version 1.7.*)
- All files must be executable

## Usage

To use the script, run it from the command line with the following syntax:
./nqueens.py N

Where N is an integer greater than or equal to 4 representing the size of the chessboard.

## Example

$ ./nqueens.py 4
[[0, 1], [1, 3], [2, 0], [3, 2]]
[[0, 2], [1, 0], [2, 3], [3, 1]]
